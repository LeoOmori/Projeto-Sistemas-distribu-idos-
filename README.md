# Projeto de SD

  O [Projeto] tem como objetivo criar um sistema distribúido os os usuário poderão buscar, salvar e ranquear piadas de um um banco de dados de piadas.
  
## Tecnologias
- NodeJs
- Nats
- Typesense
- Reactjs
